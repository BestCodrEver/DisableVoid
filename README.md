## DisableVoid
A PocketMine-MP Plugin that disables void on your server


# Features 

- Simply disables voids on your server


# Planned features

- [X] Configurable option for which worlds void should be disabled.


# BugReport

You found a bug or a plugin issue?

- Contact support in the [Discord Server](https://discord.gg/jWFB56RqUN)

- Create an [Issue](https://github.com/Vecnavium/DisableVoid/issues/new)

- Write your bug & the issue you are having with all informations including the error so I can track the issue

- Submit the issue and be patient for a response for a solution to your problem

If you are using a modified Pocketmine-MP build/version you will not get assisted.
